專案說明
===
前端部屬到 GitHub Pages

主要技術
---
* JavaScript  
* Axios  
   JavaScript庫，用於發送 http 請求
* Vue.js
* Vuex  
   用於實現不同組件間共享狀態的工具
* Vue Router  
   Vue.js 官方提供的前端路由管理器，提供了單頁面應用(SPA)
* RWD

專案功能
---
* 會員註冊、會員登入、登出
* 查看商品、排序商品(價格或商品分類)
* 查看購物車、新增商品到購物車、增加/減少購物車中的商品數量、從購物車中移除商品
* 新增訂單
